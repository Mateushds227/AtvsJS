Minha api é sobre a série de jogos da Treyarch da franquia Call Of Duty.

Dependências do projeto : Node , express .

Ele cria automaticamente o json quando executa o index.html, apagando criando/apagando automaticamente o jogos.json.

Utilizando a insomnia dá para fazer 4 requisições http , put , get , delete , post. 

Como utilizá-lo :

inicialmente no terminal colocar "npm i express", depois de um node index.js para iniciar o jogos.json e depois iniciar o servidor e utilizar o insomnia pra fazer a requisições http
